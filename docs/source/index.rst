MyGrid: Módulo para representação de redes elétricas em Python 
==============================================================

MyGrid é um módulo escrito em python para representação de redes elétricas em Python.

Para instalar o Mygrid é simples::
	
	pip install mygrid

Para utilizar o mygrid basta importa-lo em seu script:

.. code:: python

	from mygrid import Alimentador, Chave
